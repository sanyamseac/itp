# IIIT Hyderabad Intro to Portals

A guide to IIIT Hyderabad's campus portals, services, and tools.

Built with [SvelteKit](https://svelte.dev) + [Tailwind CSS](https://tailwindcss.com).

## Dev

```sh
git clone https://github.com/sanyamseac/itp
cd itp
npm install
npm run dev
```

## Build

```sh
npm run build
```
